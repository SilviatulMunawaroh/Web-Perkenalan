Tentang Saya

Saya adalah mahasiswa aktif di Universitas Nurul Jadid, prodi Teknik Informatika. Saya memiliki minat besar dalam pemrograman, pengembangan perangkat lunak, dan dunia teknologi secara umum. Saya berdomisili di desa Alaskandang, kec. Besuk, asal sekolah saya SMKN 2 Kraksaan. Di SMKN 2 Kraksaan saya mengambil jurusan Rekayasa Perangkat Lunak yang kemudian saya lanjutkan pendidikan saya di Universitas Nurul Jadid mengambil prodi Teknik Informatika.

Motivasi Kuliah di UNUJA

Motivasi saya kuliah di UNUJA adalah karena disarankan oleh orang tua yang alasannya jarak dari rumah ke kampus lumayan dekat. Selain itu saya juga tertarik karena ada jurusan Teknik Informatika untuk melanjutkan pengetahuan saya di bidang pemograman pada sekolah saya sebelumnya.

Pengalaman dan Keterampilan

Dengan pengalaman dalam pengembangan web dan pemrograman, saya telah berkontribusi pada berbagai proyek baik sebagai pekerja lepas maupun dalam konteks akademis. Keahlian saya meliputi : pengembangan Front-End (HTML, CSS, JavaScript), pengembangan Backend (PHP, Python), dan desain UI/UX.

Impian

Saya bercita-cita ingin menjadi programmer yang menciptakan solusi teknologi inovatif. Saya akan memulai dengan mempelajari dasar-dasar pemrograman seperti HTML, CSS, dan JavaScript, lalu fokus pada pengembangan backend dan frontend melalui proyek praktis. Keterampilan pemrograman memungkinkan saya membuat aplikasi yang mengotomatiskan tugas rutin. Saya berencana mempelajari berbagai bahasa pemrograman seperti Python, Java, dan C++ serta mengembangkan soft skills seperti manajemen waktu dan komunikasi. Saya juga akan mencari mentor dan bergabung dengan komunitas pemrograman untuk mendapatkan bimbingan. Dalam jangka panjang, saya ingin berkontribusi pada proyek sumber terbuka dan memimpin tim pengembangan. Saya percaya konsistensi dan ketekunan adalah kunci untuk mencapai tujuan dan memberikan dampak positif di dunia teknologi.
